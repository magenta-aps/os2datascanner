Modal:

Match the href on the link with the id of the modal, and add a data-modal="modal:open" attribute to the link.

<a href="#modal-target" data-modal="modal:open" class="button">Show Modal</a>
<div id="modal-target" class="modal" role="dialog" aria-labelledby="create-rule-modal-title" aria-hidden="true>...</div>

If opening a iframe in a modal, add a data-src="" attribute to the link as well with the src of the iframe
